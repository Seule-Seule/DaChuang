## 背景 ##
 大二那年还没有人生目标，只是因为一腔热血加入到大创项目中来，在整个过程中能力得到了锻炼、也积攒了一些项目开发的经验。虽然后期忙的不太有时间做大创，但还是认真站完了最后一班岗。希望后来的学弟学妹能用心做好每一个项目，在此备份整个项目以作参考！
